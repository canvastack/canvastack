## How to Add a New Version

Setiap kali mau naik versi, buat file baru di `docs/CHANGELOG/`:

```
docs/CHANGELOG/CHANGELOG-{versi}.md
```

Contoh:
- `CHANGELOG-0.2.0-alpha.md`
- `CHANGELOG-1.0.0-beta.md`
- `CHANGELOG-1.0.0.md`

Kemudian tambahkan baris baru di tabel **Releases** di atas.
